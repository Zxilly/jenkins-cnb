Disabled by default. When enabled, a verified pull_request.update or pull_request.target event (or a base pull-request event with an explicit update/synchronize action) aborts only older running builds of this job for the same CNB server, repository, pull-request ref, and a different source revision. Push, comment, review, other-job, and other-pull-request builds are not interrupted. Jenkins records an auditable interruption cause on every aborted build.
